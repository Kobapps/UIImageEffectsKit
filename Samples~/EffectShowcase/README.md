# Effect Showcase

Run **Tools ▸ UI Image Effects Kit ▸ Build Effect Showcase** to drop a grid of
stacked-effect examples into the open scene (plain, outline, double outline, drop shadow, glow,
multi-glow neon, combined, and crisp face), built from the sprites in `Sprites/`.

Each cell creates an `SDFImage`, embeds the field into the sprite's texture, and applies a
different effect stack — a quick tour of what the kit can do.
